Result Data

When the batch job is completed the batch job status page will include a 'Results' tab with a list of the result files. The list includes the following columns.

Name Description
Batch Job Result Type

The type of result file. The following types are supported.

  • structuredResultData
  • errorResultData
  • opaqueResultData
Request Sequence Number

For opaqueResultData the index of the original request that the file was generated for. Requests that caused an error will not have a result file. A description of the error can be obtained from the errorResultData file.
Result Data Content Type

The MIME type of the contents of the file.
Download

A button to download the file.
