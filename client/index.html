Client API Developer Guide

The Concurrent Processing Framework (CPF) provides a REST web service API that can be used to develop client applications in Java, Python or another programming language to submit batch jobs to the CPF and to download the results of those batch jobs. The API can also be accessed directly in a web browser.

Accessing the Concurrent Processing Framework

Concurrent Processing Framework is a REST web service enabled application. The web service can be used by end users using a web browser. In this mode the service returns styled HTML pages that can be used without any programming. The web service can also be accessed using a programming language using the REST Web Service API. A Java Client API is provided. Other programming languages can also access the REST Web Service API using custom code.

Security

Access to the REST web service API is limited to authorized users.
