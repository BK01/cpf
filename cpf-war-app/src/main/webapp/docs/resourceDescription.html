Resource List

The resource list file format is used to describe a resource and any child resources that are children of the resource.

The document contains a single resource object with the following fields.

Attribute Description
resourceUri The URI to the resource.
title The title of the resource.
resources An array of resource objects that can be accessed from this resource.
<customAttribute> A custom attribute specific to that resource.
